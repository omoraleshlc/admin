Files in this folder are provided for the operation of the dxExporter widget. The use of this widget requires tuning up additional tools. For more information on utilizing dxExporter, refer to the following links:
 
- http://js.devexpress.com/Documentation/Howto/Data_Visualization/Common/Widget_Export/ - "Widget Export" guide going over the whole process of exporting a widget
 
- http://js.devexpress.com/Documentation/ApiReference/Data_Visualization_Widgets/dxExporter/ - dxExporter Reference providing detailed information about the widget options and methods
